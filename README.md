# 📊 Netflix Dashboard (Power BI)

This project presents an interactive **Netflix data analysis dashboard** built using Power BI. The dashboard provides insights into Netflix content, helping understand trends in movies and TV shows.

---

## 🔍 Key Features
- 📺 Analysis of **Movies vs TV Shows distribution**
- 🌍 Content breakdown by **genre, country, and release year**
- ⭐ Visualization of **ratings and duration**
- 🗺 Identification of **top contributing countries**
- 📈 Trends in content addition over time

---

## 🛠 Tools & Technologies
- Power BI  
- Data Cleaning & Transformation  
- Data Visualization  

---

## 📈 Insights Gained
- 🎬 Majority of content consists of movies over TV shows  
- 📅 Significant growth in content addition after 2015  
- 🎭 Popular genres and ratings patterns identified  

---

## 🚀 Purpose
This project demonstrates skills in **data visualization, storytelling, and business insights generation**, making it useful for data analyst roles.
